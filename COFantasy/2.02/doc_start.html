Chroniques Oubliées Fantasy

Documentation du script COFantasy sous Roll20

Ce document a pour but de vous aider à installer et configurer le script COFantasy sur le site Roll20.net.

Prérequis:

  • Un compte Pro sur Roll20 afin de pouvoir intégrer des scripts.
  • Un niveau de connaissance minimum sur Roll20.

    Nous ferons de notre mieux pour détailler au mieux les procédures mais notre but ici n'est pas de vous expliquer le fonctionnement de Roll20 et nous partirons du principe que vous avez des notions sur ce dernier.
    Si ce n'est pas le cas, nous vous invitons tout d'abord à vous familiariser avec l'outils et savoir notamment créer/utiliser :

    • des Macros
    • des Attributs et des Capacités des personnages
  • Feuille de Personnages : Chroniques Oubliées Fantasy (French) par Nata
  • Le script Vector math pour certains calculs de distance et de collision (fait partie de la librairie de script Roll20 qui s'installe en un clic).
Attention, à cause d'un bug chez Roll20, il n'est pas possible d'installer normalement le script depuis la librarie Roll20. Il faut utiliser la méthode Import et pas le bouton Add
